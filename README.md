# 🔐 SOC with Wazuh (Docker + AI)

## 🎯 Project Goal
Build a Security Operations Center (SOC) using Wazuh to monitor logs, detect attacks, and automate response using AI.

## ⚙️ Technologies Used
- Wazuh
- Docker
- Linux
- Python (AI Automation)

## 🧠 Features
- Log monitoring and analysis
- Detection of brute force attacks
- Real-time alerts
- Automated response using Python script

## 🏗️ Architecture
- Wazuh Server (Docker)
- Wazuh Agent (Linux machine)
- Dashboard for monitoring

## 🚀 How to Run
1. Install Docker
2. Deploy Wazuh using Docker
3. Connect agents
4. Monitor logs and alerts

## 📌 Future Improvements
- Add AI script for smart detection
- Improve automation
- Add attack simulation
